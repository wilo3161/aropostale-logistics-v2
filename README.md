# Aeropostale Logistics System
Sistema modularizado y seguro para gestión de KPIs y Logística.
## Instalación
1. `python -m venv venv`
2. `source venv/bin/activate` (o `venv\Scripts\activate` en Windows)
3. `pip install -r requirements.txt`
4. Renombra `.env.example` a `.env` y verifica tus credenciales.
5. `streamlit run app.py`
